# KANBAN BOARD (Bubbletea Framework Golang)

## Features

### General:
- add / remove boards
- move task between boards
- rearrange boards
- custom, colored, global tags

### UI:
- UI adapts to terminal size
- shortcuts listed on the bottom of the screen
- maybe some cool ascii art (cause why not)

### Boards:
- custom titles
- scrolling content
- chose a color for each board
- add / remove tasks
- reorder tasks in board
- render tasks with description and tags across two lines?

### Tasks:
- custom titles
- add / remove tags
- description
- deadlines

### Save / Load
- save and load board state based on cwd
- load state automatically if there is a saved state in CWD
- create save state automatically if CWD is empty / has no board yet
